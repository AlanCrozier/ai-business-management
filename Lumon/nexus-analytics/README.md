# Lumon

Premium AI business analytics web application featuring a pixel-perfect "executive luxury analytics UI".

## Tech Stack
- React 19 + TypeScript
- Vite
- Tailwind CSS v4
- React Router DOM
- Chart.js + react-chartjs-2
- Lucide React icons
- Framer Motion

## Setup
1. Clone the repository.
2. Run `npm install`
3. Copy `.env.example` to `.env` and configure your keys.
4. Run `npm run dev` to start the development server.
